### View: Toggle Zen Mode (inspired by Zed)

```json
{
  "key": "shift+escape",
  "command": "workbench.action.toggleZenMode",
  ...
}
```

### Markdown: Open Preview to the Side

```json
{
  "key": "shift+cmd+v",
  "command": "markdown.showPreviewToSide",
  ...
}
```
