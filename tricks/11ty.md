Forbid Nunjucks processing inside Markdown:
```js
/* .eleventy.js */

eleventyConfig.addPreprocessor("autoRaw", "md", (data, content) => {
  // This regex looks for {{ }} or {% %} and wraps them
  return content.replace(/({[\{%].*?[\}%]})/g, "{% raw %}$1{% endraw %}");
});
```
