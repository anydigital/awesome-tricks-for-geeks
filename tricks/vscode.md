| Command | Action |
| --- | --- |
| Cmd + B | toggle sidebar |
| Ctrl + Cmd + I | toggle Chat (Copilot) |
| Cmd + K, then Z\n\n(Esc to exit) | switch to Zen mode\n\n- especially useful for side-by-side diffs\n- recommended Preferences: Open User Settings > Zen Mode: Center Layout OFF |
| Cmd + K, then V | edit Markdown with preview |
| Opt + Up/Down | move code |
| ***Terminal:*** |  |
| Ctrl + \` | toggle terminal |
| Ctrl + Opt + R | loop through recent commands |
| `code .` | open file/folder in VS Code |
